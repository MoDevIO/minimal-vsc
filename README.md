# Minimal VSC

A collection of themes and icon packs that makes your Visual Studio Code minimal.

## Description

**Minimal VSC** is an extension pack for Visual Studio Code that brings together a curated set of minimal themes and icon packs to help you create a clean, distraction-free coding environment.

## Included Extensions

### Icon Packs

- [Minimal Icon Theme](https://marketplace.visualstudio.com/items?itemName=7ute.minimal-icon-theme)
- [Material Product Icons](https://marketplace.visualstudio.com/items?itemName=PKief.material-product-icons)

### Themes

- [Minimal VSC Theme](https://marketplace.visualstudio.com/items?itemName=MoDevIO.modevio-minimal)

## Installation

1. Open the Extensions view in VS Code (`Ctrl+Shift+X`).
2. Search for `Minimal VSC`.
3. Click **Install** to add all included extensions.
